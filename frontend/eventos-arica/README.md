# Eventos Arica - Sistema de Gestión Municipal

Bienvenido al sistema de gestión de eventos para la Municipalidad de Arica. Esta aplicación permite administrar recintos, eventos, categorías y personal de manera eficiente y en tiempo real.

## 🚀 Documentación Completa
Para una guía detallada sobre la instalación, arquitectura y funcionamiento del proyecto, por favor consulta el archivo:

👉 **[DOCUMENTACION.md](./DOCUMENTACION.md)**

## 🛠 Instalación Rápida
1. Instalar dependencias: `npm install`
2. Iniciar desarrollo: `npm start`

## 🏗 Stack Tecnológico
- **Angular 21** (Standalone Components)
- **Tailwind CSS** (Styling)
- **RxJS & WebSockets** (Real-time)
- **NGX-Charts** (Statistics)
